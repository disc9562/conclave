# 会话与终端

DreamCoder 提供完整的会话管理和内置终端功能。

## 会话管理

### 多标签页

每个会话绑定一个独立的项目目录。点击标签栏可以：

- **新建会话** — 点击「+」创建新标签
- **切换会话** — 点击不同标签切换
- **关闭会话** — 点击标签「×」或右键关闭
- **拖拽排序** — 拖动标签调整顺序

### 项目目录

每个会话关联一个工作目录，AI 自动感知该目录下的文件结构。

切换项目：点击标签栏左侧的文件夹图标，选择新目录。

## 内置终端

### 启动终端

- **快捷键**: `Ctrl + ``（反引号）
- **UI**: 点击侧边栏终端图标

DreamCoder 内置 PTY，支持 PowerShell、cmd、bash（需要 WSL 或 git bash）。

### 终端功能

| 功能 | 说明 |
|------|------|
| 多终端 | 每个会话可开启多个终端实例 |
| 标签切换 | 终端栏可切换不同实例 |
| 复制粘贴 | 支持鼠标选中复制，`Ctrl+Shift+V` 粘贴 |
| 滚动历史 | 自动保留滚动历史 |

### 与 AI 交互

终端输出可以发送给 AI：
1. 选中终端内容
2. 右键「发送给 AI」或使用快捷键

AI 也能主动执行终端命令（需要审批）。

## 侧边栏面板

### 工作区面板

显示当前项目的文件树，可以：
- 展开/折叠目录
- 右键文件操作（打开、重命名、删除）
- 双击打开文件

### 终端面板

显示内置终端输出，可与主终端标签切换使用。
