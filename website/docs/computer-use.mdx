# Computer Use 模式

DreamCoder 支持 AI 直接控制电脑桌面，这就是 **Computer Use** 模式。AI 可以像人类一样操作应用、填写表单、点击按钮。

## 两种模式对比

| 特性 | 截图模式 (Vision) | UIA Tree 模式 |
|------|-------------------|---------------|
| 工作原理 | 定时截图 + 视觉理解 | 读取系统 Accessibility Tree |
| 成本 | 较高（每帧需视觉理解） | 极低（纯文本） |
| 速度 | 较慢 | 极快 |
| 权限需求 | 屏幕录制权限 | 辅助功能权限 |
| 适用场景 | 图像识别任务 | 自动化操作（推荐） |

:::tip
**推荐使用 UIA Tree 模式** — 成本低、速度快，足以满足大多数自动化需求。只有当 AI 需要"看"图片内容时才切换到截图模式。
:::

## 启用 Computer Use

1. 进入 **设置 → Computer Use**
2. 点击 **运行设置向导**，系统会自动：
   - 检测 Python 环境
   - 安装必要依赖
   - 引导授权系统权限

### 所需权限

| 权限 | 用途 | 模式要求 |
|------|------|----------|
| 辅助功能 (Accessibility) | 读取窗口、控件信息 | 两者都需要 |
| 屏幕录制 (Screen Recording) | 截取屏幕图像 | 仅截图模式 |

## 授权应用管理

Computer Use 只会操作你授权的应用。首次使用时会弹出授权面板：

1. 从已安装应用列表中选择需要授权的应用
2. 点击"批准"授权 AI 操作该应用
3. 后续可在设置中随时调整授权列表

## 常见问题

**Q: 为什么 AI 无法点击某些按钮？**
A: 确保目标应用已添加到授权列表，并且授予了系统辅助功能权限。

**Q: 两种模式如何切换？**
A: 在设置页面中找到「模式切换」选项，点击即可切换。

**Q: Computer Use 安全吗？**
A: AI 只能操作你授权的应用，所有敏感操作都会记录在日志中。支持在设置中开启审批确认模式。
